Web APIs: Acronym Soup

API?

Application

Programming

Interface

This differs somewhat from how HCI views APIs - they usually think about them in the sense of a programming language or software tool, not in the sense of the web. That's where almost all of the research on API design is focused.

API architectures

  • RPC: requests as function calls.
  • SOAP: XML. Resource-based; abstracted. Retro.
  • REST: URL/JSON (typically). Resource-based; exposed. Popular.
  • GraphQL: Field- and type-based. Fashionable.

Web APIs, for our purposes

Concept records (like a drug or a gene) are resources*
